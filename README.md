# Pass Manage
 This script save and manage the password of your accounts and also generate a password for you.
# Requirements
 * Linux distribution
 * Python 3 or greater
# Installation
```
 git clone https://github.com/msakrypton/pass-manage.git
 cd pass-manage
 python passmanage.py
```
